## how to install
1. Install [GNU Stow](https://www.gnu.org/software/stow/)
```bash
sudo pacman -S stow
```
2. Clone the repo inside of the home directory
```bash
cd ~/
git clone https://github.com/skylerfrags/dotfiles.git
```
3. Go into the dotfiles folder and run stow
```bash
cd dotfiles
stow .
```
